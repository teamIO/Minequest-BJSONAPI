BJSONAPI

This simply documents the methods we have. In the future, maybe, we might have an auto-generating script...

All methods return NULL if the method failed to execute (e.g. Illegal Arguments, etc)

MineQuest JSON API:

GETALLDETAILS

  • Syntax: minequest.getalldetails()
  • Returns: String[] of the loaded details

GETDETAIL

  • Syntax: minequest.getdetail(String name)
  • Param: name - refers to the detail name
  • Returns: QuestDetail object

GETSTATISTIC

  • Syntax: minequest.getstatistic(String playerName, String tableClassName)
  • Param: playerName - get the statistic for the player
  • Param: tableClassName - the full class name for the statistic, e.g. com.theminequest.MineQuest.API.Tracker.QuestStatistic
  • Returns: the Statistic that was requested for the player

GETQUESTS

  • Syntax: minequest.getquests(String playerName, int status)
  • Param: playerName - get player quests
  • Param: status - what type quests (0: GIVEN, 1: COMPLETED, 2: INPROGRESS)
  • Returns: String[] list of quests the player has

GETCOMPLETEDDETAILS

  • Syntax: minequest.getcompleteddetails(String playerName)
  • Param: playerName - get player's completed quests
  • Returns: Map of the completed quests, with its respective completion date.

HASQUEST

  • Syntax: minequest.hasquest(String playerName, String questName)
  • Param: playerName - check if said player has quest
  • Param: questName - quest to check for
  • Returns: int that represents status (0: GIVEN, 1: COMPLETED, 2: INPROGRESS, 3: UNKNOWN)

GIVEQUEST

  • Syntax: minequest.givequest(String playerName, String questName)
  • Param: playerName - player to give quest to
  • Param: questName - quest to give
  • Returns: "GOOD" if passed; "FAIL:[exception message]" if failed.

DROPQUEST

  • Syntax: minequest.dropquest(String playerName, String questName)
  • Param: playerName - player to drop quest from
  • Param: questName - quest to drop
  • Returns: "GOOD" if passed; "FAIL:[exception message]" if failed.

COMPLETEQUEST

  • Syntax: minequest.completequest(String playerName, String questName)
  • Param: playerName - player to complete quest for
  • Param: questName - quest to complete
  • Returns: "GOOD" if passed; "FAIL:[exception message]" if failed.
